# MC Collaborative Learning App

A __mobile application__ that connects students with _study partners_ or _groups_ based on __subject__, __topic__, and __schedule__. It aims to make collaborative learning easier by matching students with similar academic needs and providing tools to organize __study sessions__. The app promotes ***teamwork***, ***improves learning outcomes***, and ***builds a stronger campus community***.

## Features
- __Account Creation__
- __Student Dashboard__
- __Messaging__
- __Edit or Create To-Do-List__
- __Import Class Schedule__
